# 📘 Assignment: Data Structures in Python

## 🎯 Objective

Explore and practice using Python's built-in data structures: lists, dictionaries, sets, and tuples.

**Skills practiced:** Data structure operations, indexing, iteration, manipulation, and choosing appropriate structures for tasks

## 📝 Tasks

### 🛠️ Implement Data Structure Operations

#### Description
Create a Python program that demonstrates various operations on lists, dictionaries, sets, and tuples through practical examples.

#### Requirements
Completed program should:

- Create and manipulate a list (add, remove, sort elements)
- Use a dictionary for key-value storage and retrieval
- Perform set operations (union, intersection, difference)
- Work with tuples for immutable data
- Demonstrate iteration over each data structure
- Show practical use cases for each (e.g., shopping list for lists, contact book for dictionaries)
