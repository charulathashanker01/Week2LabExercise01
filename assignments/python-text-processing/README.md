# 📘 Assignment: Python Text Processing

## 🎯 Objective

Work with strings, file I/O, and text manipulation in Python to build a text processing program.

**Skills practiced:** String methods, file reading/writing, text parsing, basic data structures

## 📝 Tasks

### 🛠️ Build a Text Processor

#### Description
Create a Python program that reads text from a file, performs various manipulations, and writes the results to another file.

#### Requirements
Completed program should:

- Read text from an input file (e.g., 'input.txt')
- Count the total number of words in the text
- Replace all occurrences of a specific word (e.g., 'old' with 'new')
- Convert the text to uppercase or lowercase
- Write the processed text and statistics to an output file (e.g., 'output.txt')
- Handle file not found errors gracefully
